# Enrich-Sentinel-Incident-HYAS-Insight-IP-Dynamic-DNS
author: Paul van Gool, HYAS Infosec

This playbook uses the HYAS Insight connector to automatically enrich incidents generated by Sentinel with recent Dynamic DNS information. You need a valid subscription in order to use the connector and playbook. Learn more about the integration via the [connector documentation](https://docs.microsoft.com/connectors/hyasinsight/) or visit [HYAS Insight](https://www.hyas.com/contact) to request a trial key.
