# Get-VirusTotalURLAnalyses
author: Nicholas DiCola

This playbook will take each URL entity and query VirusTotal for analysis (https://developers.virustotal.com/v3.0/reference#urls).  You will need to register to thier community for an API key.  If the scan has any malicous or suspicous hits it will add a comment to the incident.
